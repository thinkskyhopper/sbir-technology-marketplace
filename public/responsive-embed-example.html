Responsive SBIR Tech Marketplace Widget

This example shows how to embed the SBIR Tech Marketplace widget in a responsive way that works perfectly on all devices including mobile phones and tablets.

Key Features:

  • Fully responsive design that adapts to any screen size
  • Mobile-optimized layout with touch-friendly interactions
  • Automatic height adjustments for different breakpoints
  • Maintains all functionality across devices

How to Use This Responsive Embed

Copy the following HTML and CSS code to embed the responsive widget on your website:

HTML:

<div class="responsive-widget-container"> <iframe src="https://sbir-technology-marketplace.lovable.app/embed.html" class="responsive-widget-iframe" frameborder="0" scrolling="no" title="SBIR Tech Marketplace Widget"> </iframe> </div>

CSS:

/* Responsive iframe wrapper */ .responsive-widget-container { width: 100%; max-width: 600px; margin: 0 auto; position: relative; } .responsive-widget-iframe { width: 100%; height: 450px; border: 1px solid #ddd; border-radius: 8px; display: block; } /* Mobile adjustments */ @media (max-width: 768px) { .responsive-widget-iframe { height: 400px; } } @media (max-width: 480px) { .responsive-widget-iframe { height: 350px; border-radius: 6px; } }

Alternative: Simple Responsive Embed

For a simpler approach, you can just use this single line with inline styles:

<iframe src="https://sbir-technology-marketplace.lovable.app/embed.html" style="width: 100%; max-width: 600px; height: 450px; border: 1px solid #ddd; border-radius: 8px;" frameborder="0" scrolling="no" title="SBIR Tech Marketplace Widget"> </iframe>
